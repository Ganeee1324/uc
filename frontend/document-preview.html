Analisi Matematica I

Appunti delle Lezioni

Prof. Giuseppe Rossi - Università di Milano

1. Introduzione alle Funzioni

Le funzioni matematiche rappresentano uno dei concetti fondamentali dell'analisi matematica. In questa sezione introduciamo le definizioni base e le proprietà principali.

1.1 Definizione di Funzione

Una funzione f: A → B è una regola che associa ad ogni elemento x ∈ A esattamente un elemento f(x) ∈ B.

1.2 Dominio e Codominio

Il dominio di una funzione è l'insieme di tutti i valori per cui la funzione è definita, mentre il codominio è l'insieme di arrivo della funzione.

Esempio: f(x) = x² + 2x - 1

Dominio: ℝ (tutti i numeri reali)

2. Limiti di Funzioni

Il concetto di limite è cruciale per comprendere il comportamento delle funzioni in prossimità di determinati punti.

2.1 Definizione di Limite

Si dice che lim(x→a) f(x) = L se per ogni ε > 0 esiste δ > 0 tale che se 0 < |x - a| < δ allora |f(x) - L| < ε.

Teorema: Se il limite esiste, allora è unico.

2.2 Proprietà dei Limiti

  • Linearità: lim[af(x) + bg(x)] = a·lim f(x) + b·lim g(x)
  • Prodotto: lim[f(x)·g(x)] = lim f(x) · lim g(x)
  • Quoziente: lim[f(x)/g(x)] = lim f(x) / lim g(x) (se lim g(x) ≠ 0)

2.3 Esempi Pratici

Calcoliamo alcuni limiti fondamentali:

lim(x→0) sin(x)/x = 1

lim(x→∞) (1 + 1/x)ˣ = e

3. Derivate

La derivata di una funzione rappresenta il tasso di variazione istantaneo della funzione stessa.

3.1 Definizione di Derivata

La derivata di f(x) nel punto a è definita come:

f'(a) = lim(h→0) [f(a+h) - f(a)]/h

3.2 Regole di Derivazione

  • Regola della potenza: (xⁿ)' = n·xⁿ⁻¹
  • Regola del prodotto: (f·g)' = f'·g + f·g'
  • Regola del quoziente: (f/g)' = (f'·g - f·g')/g²
  • Regola della catena: (f∘g)' = f'(g(x))·g'(x)

3.3 Interpretazione Geometrica

La derivata in un punto rappresenta la pendenza della retta tangente alla curva in quel punto.

Esempio: f(x) = x³ - 2x + 1

f'(x) = 3x² - 2

Nel punto x = 1: f'(1) = 3(1)² - 2 = 1

4. Integrali

L'integrale rappresenta l'operazione inversa della derivazione e ha molteplici interpretazioni geometriche e fisiche.

4.1 Integrale Indefinito

L'integrale indefinito di f(x) è l'insieme di tutte le primitive di f(x):

∫ f(x) dx = F(x) + C

dove F'(x) = f(x) e C è una costante arbitraria

4.2 Integrale Definito

L'integrale definito rappresenta l'area sottesa dalla curva f(x) nell'intervallo [a,b]:

∫ₐᵇ f(x) dx = F(b) - F(a)

4.3 Tecniche di Integrazione

  • Integrazione per parti: ∫ u dv = uv - ∫ v du
  • Sostituzione: ∫ f(g(x))g'(x) dx = ∫ f(u) du
  • Frazioni parziali per funzioni razionali

4.4 Teorema Fondamentale del Calcolo

Questo teorema stabilisce la connessione tra derivazione e integrazione.

5. Serie e Successioni

Le serie e le successioni sono strumenti fondamentali per approssimare funzioni e risolvere equazioni complesse.

5.1 Successioni

Una successione è una funzione definita sui numeri naturali: a: ℕ → ℝ

aₙ = f(n) per n ∈ ℕ

5.2 Convergenza di Successioni

Una successione {aₙ} converge a L se:

lim(n→∞) aₙ = L

5.3 Serie Numeriche

Una serie è la somma dei termini di una successione:

Σ(n=1 to ∞) aₙ = a₁ + a₂ + a₃ + ...

5.4 Criteri di Convergenza

  • Criterio del confronto
  • Criterio della radice
  • Criterio del rapporto
  • Criterio di Leibniz per serie alternate
